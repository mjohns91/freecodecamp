Tim Berners-Lee

The man who invented the world wide web

Timeline of Sir Berners-Lee's professional life:

  • 1955 - Tim is born in London, England
  • 1976 - Graduates from The Queen's College in Oxford, with a B.A. in physics
  • 1978 - Joined D.G. Nash, where he helped create typesetting software for printers
  • 1980 (June) - Worked as a contractor at CERN and built his prototype ENQUIRE system, the precursor to the web
  • 1980 (Dec) - Ran the technical side of John Poole's Image Computer Systems, Inc.
  • 1984 - Returned to CERN as a fellow
  • 1989 - Wrote his proposal for joining hypertext with the Internet
  • 1990 - Published the first web site, which described the project itself and explained how people could use a browser and set up a web server / web site
  • 1994 - Founded the W3C at MIT, which focused on developing standards and recommendations to improve the Web
  • 2009 - Founded the World Wide Web Foundation in order to campaign to "advance the Web to empower humanity by launching transformative programs that build local capacity to leverage the Web as a medium for positive change"

"Creating the web was really an act of desperation, because the situation without it was very difficult when I was working at CERN later. Most of the technology involved in the web, like the hypertext, like the Internet, multifont text objects, had all been designed already. I just had to put them together. It was a step of generalising, going to a higher level of abstraction, thinking about all the documentation systems out there as being possibly part of a larger imaginary documentation system."

-- Tim Berners-Lee
